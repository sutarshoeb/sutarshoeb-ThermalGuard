# Research Assets

This folder is the Git-friendly research surface for the repository.

It should contain:

- benchmark comparison tables
- curated experiment summaries
- figures or diagrams small enough for Git
- references to larger notebooks, datasets, and model artifacts that stay outside version control

The larger historical work remains in the repository root for now:

- `notebook/`
- `scripts/`
- `src/`
- `results/`
- `models/`

As the project matures, we can continue curating this folder into the clean public research story.
